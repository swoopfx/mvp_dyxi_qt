# ABC & 123 Fun - Dyslexia Friendly Game

A colorful, engaging educational game designed specifically for children with dyslexia (ages 3-8).

## Features
- **Dyslexia-Friendly Typography:** Uses the OpenDyslexic font for improved readability.
- **High-Readability Colors:** Warm, low-stress background colors (cream/peach) instead of harsh white.
- **Multiple Levels:**
  - **Level 1 (Recognition):** Identify letters and numbers in a fun grid.
  - **Level 2 (Matching):** Match letters to objects starting with that letter.
  - **Level 3 (Tracing):** Practice letter formation with a guided tracing canvas.
- **Audio Feedback:** Encouraging voiceovers and sound effects for positive reinforcement.
- **Graphical Assets:** Custom-generated, child-friendly 3D icons and characters.

## Requirements
- Python 3.11+
- PySide6 (Qt for Python)

## How to Run
1. Ensure you have the required dependencies:
   ```bash
   pip install PySide6
   ```
2. Run the game:
   ```bash
   python main.py
   ```

## Design Principles
- **Reduced Crowding:** Generous spacing between letters and UI elements.
- **Consistency:** Simple, predictable navigation.
- **Multi-sensory:** Combines visual cards, audio instructions, and touch/mouse interaction.
