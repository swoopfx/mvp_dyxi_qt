# Sum Swamp Adventure

A highly graphical and animated educational math game built with QML and Qt C++.

## Features
- **Vibrant Graphics**: Custom-generated swamp-themed board and characters.
- **Animated Gameplay**: Smooth character movement and interactive UI.
- **Difficulty Levels**:
    - **Tadpole**: Addition only, numbers 1-5.
    - **Frog**: Addition and subtraction, numbers 1-6.
    - **Gator**: Addition and subtraction, numbers 1-12.
- **Comprehensive Metrics**:
    - Mental math speed and accuracy.
    - Error evaluation (off-by-one, sign errors).
    - Concentration tracking.
- **JSON Logging**: Every event is logged for educational analysis.
- **Complementary Tasks**: Number sense challenges triggered during gameplay.

## Project Structure
- `cpp/`: C++ backend for heavy logic and data processing.
- `qml/`: QML files for the frontend UI and animations.
- `assets/`: Images and audio files.
- `data/`: JSON logs and metrics.

## How to Run
1. Ensure you have Qt 5.15+ or Qt 6 installed.
2. Open the project in Qt Creator.
3. Build and run.
